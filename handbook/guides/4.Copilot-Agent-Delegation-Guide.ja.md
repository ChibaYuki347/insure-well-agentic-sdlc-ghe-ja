# Copilot エージェント委任ガイド

## 概要

**作業をどこから始めるか**、そして **どの程度インタラクティブに進めたいか** に応じて、適切な委任モードを選択してください。

---

## クイック比較

| エージェント | 場所 | 適した用途 | スタイル |
|---|---|---|---|
| **VS Code ローカルエージェント** (Agent Mode) | エディター内 | インタラクティブなコーディング、デバッグ、リファクタ | ワークスペースに対するライブ編集 |
| **VS Code クラウドエージェント** | エディター内 | 非同期での作業委任 | バックグラウンドで PR を作成 |
| **GitHub Issue → Copilot** | GitHub.com | バックログとして追跡される作業 | Issue から PR への自動化 |

---

## 3 つのモードの解説

### 1️⃣ VS Code ローカルエージェント (Agent Mode) 🛠️

**こんなとき:** コーディング中に密にインタラクティブなやり取りをしたいとき。

- ワークスペース、ファイル、ターミナル状態、未コミットの変更を把握
- デバッグ、リファクタ、コードの説明に最適
- すべてのステップを自分で操舵する

**例:** 「この Spring Boot コントローラはなぜ失敗するの?」「この関数をリファクタして」「このクレーム更新をトレースして」

---

### 2️⃣ VS Code クラウドエージェント ☁️

**こんなとき:** コーディング中にタスクを見つけて非同期で任せたいとき。

- コーディング中に発見した、独立した作業
- 自分では実装せず Copilot がバックグラウンドで実装
- 変更案または PR が成果物になる

**例:** 「このモジュールにテストを追加して」「このコンポーネントを整理して」「このヘルパーを実装して」

---

### 3️⃣ GitHub Issue → Copilot Coding Agent 📌

**こんなとき:** 作業がすでに GitHub Issue として追跡されているとき。

- 作業がチームのバックログの一部
- 開始時点からチーム全体に可視
- Issue の説明だけで実装に十分なコンテキストがある

**例:** 「Issue #42 は実装可能な状態」「このバグを Copilot にアサイン」「この機能を実装して」

---

## クイック判断ガイド

| やりたいこと | 使うべきもの |
|---|---|
| 今すぐライブでインタラクティブに支援が欲しい | **ローカルエージェント** |
| コーディング中に見つけたタスクを委任したい | **クラウドエージェント** |
| バックログの作業を追跡・実行したい | **GitHub Issue → Copilot** |
| ローカルなコンテキスト (ローカル DB、開発サーバー、未公開ファイル) が必要 | **ローカルエージェント** |
| チームの可視性とワークフロー統合が必要 | **GitHub Issue → Copilot** |

---

## 機能スナップショット

| 機能 | ローカル | クラウド | GitHub Issue |
|---|---|---|---|
| インタラクティブ性 | ✅ 優秀 | ❌ 限定的 | ❌ 限定的 |
| ローカルデバッグ | ✅ 優秀 | ❌ 弱い | ❌ 弱い |
| 非同期委任 | ⚠️ 中 | ✅ 優秀 | ✅ 優秀 |
| チームの可視性 | ❌ なし | ⚠️ ときに有り | ✅ 優秀 |
| バックログ実行 | ⚠️ 中 | ✅ 良 | ✅ 優秀 |

---

## 用語の整理

- **ローカルエージェント** = `VS Code Local Agent (Agent Mode)` = VS Code 内の Agent Mode で動く Copilot
- **クラウドエージェント** = VS Code から非同期で委任 (作業はバックグラウンドで実行)
- **GitHub Issue 委任** = GitHub.com 上で Issue を Copilot にアサインする方式

---

## 最終的な判断基準

🎯 **ローカル = 今すぐインタラクティブに。** | ☁️ **クラウド = 任せて非同期に。** | 📌 **GitHub = 追跡してチームで。**

ローカルかつインタラクティブな作業 → **ローカルエージェント**  
コーディング中に発見した作業 → **クラウドエージェント**  
すでに Issue として追跡されている作業 → **GitHub Issue → Copilot**
